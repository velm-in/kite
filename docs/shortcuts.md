# Keyboard Shortcuts

All shortcuts use `Cmd` on macOS and `Ctrl` on Windows/Linux.

---

## General

| Action | Shortcut |
|--------|----------|
| Command palette | `Cmd+K` |
| Settings | `Cmd+,` |
| New tab | `Cmd+T` |
| Close tab | `Cmd+W` |
| Next tab | `Cmd+]` or `Ctrl+Tab` |
| Previous tab | `Cmd+[` or `Ctrl+Shift+Tab` |

---

## Navigation

| Action | Shortcut |
|--------|----------|
| API Client | `Cmd+1` |
| Collections | `Cmd+2` |
| Webhooks | `Cmd+3` |
| History | `Cmd+4` |
| SSH | `Cmd+5` |

---

## Requests

| Action | Shortcut |
|--------|----------|
| Send request | `Cmd+Enter` |
| Save request | `Cmd+S` |
| Duplicate request | `Cmd+D` |
| Format JSON body | `Cmd+Shift+F` |

---

## Command palette

Press `Cmd+K` to open. Type to search across:

- Requests in all collections
- Webhook endpoints
- SSH hosts
- Actions (new tab, import, settings, etc.)
- URL history (autocomplete)

The command palette uses a Rust-powered trie for instant URL suggestions and full-text search across collections.
