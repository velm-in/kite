# Contributing to Kite

Thanks for your interest in contributing. Here's how to help.

---

## Reporting bugs

[Open a bug report](https://github.com/velm-in/kite/issues/new?template=bug_report.md) with:

- What you expected to happen
- What actually happened
- Steps to reproduce
- Your OS and Kite version (shown in Settings or status bar)
- Screenshots if relevant

---

## Requesting features

[Open a feature request](https://github.com/velm-in/kite/issues/new?template=feature_request.md) with:

- What problem you're trying to solve
- How you'd expect it to work
- Any alternatives you've considered

---

## Security vulnerabilities

Do **not** open a public issue for security vulnerabilities. Email security@velm.in instead. We'll respond within 48 hours.

---

## Code contributions

Kite is not open source at this time. The codebase is proprietary. This repository hosts release binaries, documentation, and issue tracking.

If you're interested in contributing code in the future, watch this repo for updates.

---

## Documentation

Found a typo or unclear docs? PRs to the `docs/` folder are welcome.

---

## Community
