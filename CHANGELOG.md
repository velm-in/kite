- Localhost tunneling via WebSocket relay
- SSH terminal with host management
- SFTP file browser
- Port forwarding (local and remote)
- Run-from-remote execution
- Health dashboard and alert system
- Collection sync (Pro/Team)
- Workspace collaboration (Team)
- Import from Postman, Bruno, Insomnia, OpenAPI, cURL
- Auto-updates with signed binaries
- Command palette with full-text search
- Environment variables with 5-level nesting
- Cookie jar with per-domain storage
- Request history (100 entries)
- Mock server
- Payload diff viewer
- Batch test runner
